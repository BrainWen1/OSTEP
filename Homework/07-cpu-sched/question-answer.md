4. 观察上面的测试可知，当所有作业的长度相同时，FIFO和SJF的周转时间一致，除此之外，二者的响应时间也一致。

5. 思考可得出，只有一个作业时，或者多个作业的长度全部相同并且RR的量子长度为作业长度时，SJF和RR提供相同的响应时间。

6. 结论：
    - 对于 “从短作业变成长作业” 的任务：响应时间会显著变长；
    - 对于 “原本就是长作业” 的任务：响应时间基本不变（但完成时间会变长）。

    原理：
    随着工作长度的增加，SJF 的响应时间的变化：
    - 若某个作业原本是 “短作业”（比如长度 1），当它的长度增加为 “长作业”（比如长度 5）：
        它会被排在所有更短的作业之后，响应时间从 “0（直接被调度）” 变成 “前面所有短作业的总长度”（比如前面有 2 个短作业，总长度 3，则响应时间 = 3）；
    - 若某个作业原本就是 “长作业”（比如长度 5），即使长度继续增加为 10：
        只要前面的短作业不变，它的响应时间依然是 “前面短作业的总长度”（因为它还是长作业，依然排在短作业之后）。

7. 结论：
    随着量子长度q增加，SJF的最坏响应时间呈现和q正相关的线性增长。
    
    分析：
    初始量子长度为q，对于n个作业，平均响应时间就是中间作业的响应时间或者中间两侧作业的平均值。
    给定N个作业，最坏响应时间就是 t = (n / 2) * q，此方程中 (n / 2) 为常数，那么该函数就是一个斜率为 (n + 2) 的线性函数，q为自变量，响应时间t线性增长。
